## Extra Search
This userscript exposes search fields in the /tags and /artists pages that are normally not shown to regular users.

### Installing
1. Install [ViolentMonkey](https://violentmonkey.github.io/) or another userscript manager
1. Head to [the script](https://github.com/shemingwan-wen/extra-search/raw/refs/heads/main/dist/extrasearch.user.js) to install it

### Building
1. Clone this repository
1. Run `npm install` to get all the dependencies
1. Run `npm run build` and find the build in the dist/ folder
